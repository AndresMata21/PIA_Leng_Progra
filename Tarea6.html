- Sintaxis -

Hay dos maneras de trabajar con Python, mediante el modo interactivo y mediante el modo script. En el modo interactivo, debemos activar el intérprete Python escribiendo python en la línea de comandos. Una vez puesto en marcha, ya podemos ejecutar órdenes, como la clásica print “Hello, Python!”. Así, una vez situada la línea de comandos en la carpeta donde se encuentra el script, lo ejecutamos con la orden python nombredelscript.py.
