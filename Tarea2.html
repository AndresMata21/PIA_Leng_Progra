- LENGUAJES INFORMÁTICOS -

- La Clasificación de los Lenguajes Informáticos -

Los Lenguajes Informáticos se pueden clasificar en:

  • Lenguaje de programación
  • Lenguaje de especificación
  • Lenguaje de consulta
  • Lenguaje de marcas
  • Lenguaje de transformación
  • Protocolo de comunicaciones
  • Lenguaje de sonido
  • Lenguaje gráfico
  • Pseudocódigo
  • Lenguaje de Definición de Procesos (Process Definition Language, PDL)

- Ejemplos -

Los Lenguajes Informáticos se pueden clasificar en:

  • Lenguaje de Consulta: SQL o XQuery
  • Lenguaje de Marca: XML
  • Lenguaje de Transformación: XSLT
  • Protocolo de comunicaciones: HTTP, FTP
  • Lenguaje de Programación: Java, Pearl, Python
  • Lenguaje de Especificación: Alloy, Estelle, Lotos
