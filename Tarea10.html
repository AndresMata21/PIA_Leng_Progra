Tarea #10
- GÖDEL -

- Símbolos -

<-, /, \, &, =, ~, :, *,

- Sintaxis -

Conjunto de proposiciones lógicas escritas como cláusulas de Horn que permiten inferir el valor de verdad de nuevas proposiciones, permitiendo ampliar la base de conocimientos, a la vez que son utilizadas para definir el dominio del problema.

- Esqueleto de Programas -

  • MODULE nombre
  • IMPORT librerías
  • PREDICATE función.

- Aplicación descargada para Editar y Ejectutar -

SICStus

- Paradigma al que Pertenece -

Programación lógica.

- Ejemplo de Programa Escrito -

  • MODULE MCD
  • IMPORT Integers
  • PREDICATE Mcd : Integer * Integer * Integer.
  • Mcd(i,j,d) <-
  • DivisorComun(i,j,d) &
  • ~ SOME [e] (DivisorComun(i,j,e) & e > d).
  • PREDICATE DivisorComun : Integer * Integer * Integer.
  • DivisorComun(i,j,d) <-
  • IF (i = 0 \/ j = 0)
  • THEN
  • d = Max(Abs(i),Abs(j))
  • ELSE
  • 1 =< d =< Min(Abs(i),Abs(j)) &
  • i Mod d = 0 &
  • j Mod d = 0.
